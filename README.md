# Auth System

Auth system built with React (Vite + TypeScript), Tailwind CSS, Node.js (Express + TypeScript), and PostgreSQL.

## 🔧 Stack
- React + Vite + TS
- Tailwind CSS
- Node.js + Express
- PostgreSQL

## 🚀 Setup

### Backend
- cd ./backend
- npm install
- .env with: USER_NAME, DATABASE, PSWD, DB_PORT, JWT_SECRET
- npm run dev


### Frontend

- cd ./frontend
- npm install
- npm run dev


## 📁 Structure

auth-system/
├── frontend/ 
└── backend/ 


## ✅ Features
- JWT Auth
- Type-safe
- Tailwind UI
